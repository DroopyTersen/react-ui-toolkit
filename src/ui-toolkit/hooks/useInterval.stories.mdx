# useInterval

Lets you perform execute a function on the specified interval. You can turn it off by passing a `delay` of `0`.

```javascript
// Ex: Poll for new data every 5 seconds
useInterval(() => {
  checkForChanges();
}, 5000);
```

## Import

```tsx
import { useInterval } from "~ui-toolkit/hooks/useInterval";
```
