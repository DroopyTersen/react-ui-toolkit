# useGrid

A hook to help you create and pass props to grid components. Helps to deal with things like paging the items.

```javascript
// You can grab the whole grid object
let grid = useGrid(items, { pageSize: 25 });
// Or you can destructure
let { showingItems, paging, stats } = useGrid(items, { pageSize: 25 });
```

### Import

```tsx
import { useGrid } from "~ui-toolkit/hooks/useGrid";
```
