# usePagingStats

A hook to calculate paging statistics (start/end range, total pages)

```javascript
let { totalItems, totalPages, start, end } = usePagingStats(100, 25, 1);
```

## Import

```tsx
import { usePagingStats } from "~ui-toolkit/hooks/usePagingStats";
```
