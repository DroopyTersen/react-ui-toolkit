# useSorting

A hook to help you manage sorting. Helps to deal with things like wrapping around back
to page 1 when you reach the max number of pages.

```javascript
// You can grab the whole sorting object
let sorting = useSorting(items, { sortKey: "name", sortDir: "asc" });
// Or you can destructure
let { showingItems, onSort, sortKey, sortDir } = useSorting(items, {
  sortKey: "name",
  sortDir: "asc",
});
```

### Import

```tsx
import { useSorting } from "~ui-toolkit/hooks/useSorting";
```
