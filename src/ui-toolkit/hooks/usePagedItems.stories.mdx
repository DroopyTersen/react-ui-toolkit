# usePagedItems

A hook to help you manage only showing items on the current page.

```javascript
// Pass in the full item set and the number of items per page
let [pagedItems, paging] = usePagedItems(allItems, 10);

return (
  <>
    <Grid>
      {pagedItems.map((item) => (
        <Card key={item.id} title={item.name} />
      ))}
    </Grid>
    <div>
      <button disabled={paging.currentPage <= 1} onClick={paging.goBack}>
        Prev
      </button>
      <button disabled={paging.currentPage >= paging.totalPages} onClick={paging.goForward}>
        Next
      </button>
    </div>
  </>
);
```

### Import

```tsx
import { usePagedItems } from "~ui-toolkit/hooks/usePaging";
```
